Kerala Stats

Kerala COVID-19 API

A database and API for COVID-19 statstics for Kerala. Data is updated periodically and automatically using Github Actions.

API

# Data URL
1 Time-series District-wise Histories of COVID-19 stats https://keralastats.coronasafe.live/histories.json
2 Latest District-wise COVID-19 stats https://keralastats.coronasafe.live/latest.json
3 Latest Summary of COVID-19 stats https://keralastats.coronasafe.live/summary.json
4 Time-series Test Reports https://keralastats.coronasafe.live/testreports.json
5 Time-series Hotspots Histories https://keralastats.coronasafe.live/hotspots_histories.json
6 Latest Hotspots https://keralastats.coronasafe.live/hotspots.json
7 Time-series District-wise Histories of vaccination stats https://keralastats.coronasafe.live/vaccination_histories.json
8 Latest District-wise vaccination stats https://keralastats.coronasafe.live/vaccination_latest.json
9 Latest Summary of vaccination stats https://keralastats.coronasafe.live/vaccination_summary.json
